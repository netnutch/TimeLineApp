# 一个时间管理类App
设置开始时间和结束时间，在规定的时间内制定清单，再也不怕刷手机忘记正事了, 因为悬浮窗的任务时时刻刻提醒着你。
赶快试试吧

# 需要权限：
- 悬浮窗

# 特性:
    将数据以悬浮窗形式显示
